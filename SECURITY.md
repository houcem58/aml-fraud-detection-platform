# Security and Scope

## Repository Scope

This repository is a **portfolio showcase** for the AML Fraud Detection Platform.

**What is included:**
- Architecture documentation and system design
- Evaluation methodology and benchmark results
- Demo scripts using synthetic transaction data
- Business case and use case documentation

**What is NOT included:**
- Production source code (`src/`)
- Trained model weights (XGBoost, HybridGAT)
- Docker Compose and infrastructure configuration
- Test suite and sprint artifacts
- Real transaction data of any kind

The full implementation is maintained in a private repository.

## Reporting Issues

If you find a documentation error or security concern in this showcase repository,
open a GitHub issue or contact houcem0508@gmail.com.

## Data Privacy

The demo scripts use **entirely synthetic data** — randomly generated transaction
records with no connection to any real financial institution, customer, or account.
No real transaction data is stored anywhere in this repository.
